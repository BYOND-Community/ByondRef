

 Repop proc (world)
--------------------




**Format:** 


 Repop()
 


**Format:** 

**Format:**

 Repop()



**Default action:** 


 Reload the obj and mob instances defined in the world map. This
 "repopulates" a world to its initial state. Only objects that were
 destroyed will be recreated.
