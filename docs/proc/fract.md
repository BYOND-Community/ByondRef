**Args:** 

**Args:**

 A: A number.


 Returns the fractional part of the number A, with the same sign. This is
everything after the decimal point.



### 
 Example:



 usr << fract(1.45) // outputs 0.45

usr << fract(-1.45) // outputs -0.45
