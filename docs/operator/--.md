**Format:** 

**Format:**

 --A // pre-decrement
 
 A-- // post-decrement
 


 A-- // post-decrement


 The pre-decrement has the value (A-1) and the effect of subtracting 1
from A.




 The post-decrement has the value (A) and has the effect of subtracting 1
from A.



### 
 Example:



 var/A = 0
world << "A-- = [A--]" // outputs "A = 0"
world << "--A = [--A]" // outputs "A = -2"
